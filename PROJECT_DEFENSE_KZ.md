# AI Teacher Test Generator жобасын қорғау мәтіні

## Жобаның атауы

AI Teacher Test Generator

## Жобаның мақсаты

Бұл жобаның мақсаты - мұғалімге немесе болашақ информатика педагогіне сабаққа, қайталау жұмысына, үй тапсырмасын тексеруге және қорытынды бағалауға арналған тест тапсырмаларын автоматты түрде құруға көмектесу.

## Неліктен бұл жоба қажет

Мұғалімнің уақытын көп алатын жұмыстардың бірі - сапалы тест құрастыру. Әсіресе әр түрлі тақырыпқа, әр түрлі күрделілік деңгейіне және әр түрлі форматқа сай тапсырма дайындау қиын. Бұл жоба сол мәселені шешуге бағытталған.

Жоба келесі жағдайларда пайдалы:

- жаңа тақырыпты бекіту кезінде;
- қайталау сабағында;
- қалыптастырушы бағалауда;
- үй тапсырмасын тексеруде;
- емтиханға дайындықта;
- белгілі бір құжат немесе оқу материалы негізінде тест құрастыруда.

## Педагогикалық құндылығы

Бұл қосымша болашақ информатика педагогіне пайдалы, себебі:

- білім беру процесінде жасанды интеллектіні тиімді қолдануды көрсетеді;
- оқу мақсатына сай тест құрастыруды жеңілдетеді;
- тапсырмаларды күрделілік деңгейі бойынша бейімдеуге мүмкіндік береді;
- мұғалімге дайын материалды өңдеп, түзетуге жағдай жасайды;
- үш тілде жұмыс істей алады: ағылшын, орыс, қазақ.

## Жобаның жұмыс істеу логикасы

1. Пайдаланушы тақырыпты енгізеді немесе PDF, DOCX, TXT файлын жүктейді.
2. Қажетті параметрлерді таңдайды:
   - сұрақ саны;
   - күрделілік деңгейі;
   - тест түрі;
   - тіл;
   - сынып деңгейі;
   - сабақ кезеңі;
   - бағалау мақсаты;
   - оқу мақсаты.
3. Қосымша Groq cloud AI сервисіне сұраныс жібереді.
4. AI құрылымды JSON форматында тест жасайды.
5. Қолданушы әр сұрақты тікелей интерфейсте өңдей алады.
6. Әр сұраққа түсіндірме және skill tag беріледі.
7. Жүйе сапа есебін шығарады:
   - қате бар ма;
   - қайталанатын сұрақ бар ма;
   - дұрыс жауап таңдалған ба;
   - экспортқа дайын ба.
8. Variant A, Variant B, Variant C автоматты түрде жасалады.
9. Student mode арқылы оқушы тест тапсыра алады, ал жүйе нәтижені автоматты тексереді.
10. Analytics dashboard нәтижені талдайды.
11. Сұрақтар question bank ішінде бөлек сақталады.
12. Барлық нұсқалар SQLite жергілікті тарихына сақталады және қажет болса Supabase бұлтына синхрондалады.

## Cloud AI қалай қосылған

Жоба нақты бұлттық ЖИ моделін қолданады.

- API интеграциясы `ai_generator.py` файлында жүзеге асқан.
- API кілті `.env` файлы арқылы беріледі.
- Қолданылатын айнымалы: `GROQ_API_KEY`
- Қолданылатын модель: `openai/gpt-oss-20b`
- Жауаптың дұрыс құрылымда келуі үшін strict JSON schema пайдаланылады.

## Қорғауда көрсетуге болатын код бөліктері

1. `ai_generator.py`
   - API клиентін құру
   - Groq API-ге сұраныс жіберу
   - structured output schema

2. `app.py`
   - Streamlit интерфейсі
   - файл жүктеу
   - сұрақтарды өңдеу
   - student/teacher export
   - quality report

3. `document_loader.py`
   - PDF, DOCX, TXT мәтінін оқу

4. `storage.py`
   - SQLite тарихын сақтау
