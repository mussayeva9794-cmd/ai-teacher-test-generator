# Бағалау критерийлеріне сәйкестік

## 1. Жобаның тақырыпқа сәйкестігі - 10 балл

Жоба білім беру саласына арналған. Негізгі мақсаты - мұғалімге және болашақ информатика педагогіне тест тапсырмаларын жылдам дайындауға көмектесу.

## 2. Педагогикалық негіздеме - 10 балл

Жоба келесі оқу жағдайларында қолданылады:

- қалыптастырушы бағалау;
- сабақты бекіту;
- қайталау;
- емтиханға дайындық;
- үй тапсырмасын тексеру.

Қосымшада оқу мақсаты, сабақ кезеңі, бағалау мақсаты, сынып деңгейі сияқты педагогикалық параметрлер бар. Сонымен бірге student mode және analytics педагогикалық шешім қабылдауға көмектеседі.

## 3. Python кодтың сапасы - 15 балл

Код бірнеше модульге бөлінген:

- `app.py`
- `ai_generator.py`
- `document_loader.py`
- `storage.py`
- `quality.py`

Бұл құрылым кодты түсінікті және қолдауға ыңғайлы етеді.

Қосымша модульдер:

- `analytics.py`
- `variants.py`
- `cloud_sync.py`

## 4. Интерфейс сапасы - 10 балл

Интерфейс Streamlit негізінде жасалған:

- форма қарапайым;
- параметрлер анық орналасқан;
- тарих панелі бар;
- профиль панелі бар;
- question bank бар;
- сапа есебі көрінеді;
- student mode бар;
- analytics dashboard бар;
- экспорт батырмалары түсінікті.

## 5. Бұлттағы ЖИ моделін қосу - 20 балл

Жоба нақты Groq cloud AI сервисін қолданады және қосымша түрде Supabase cloud backup қолдайды.

- `GROQ_API_KEY` `.env` арқылы сақталады;
- API сұранысы `ai_generator.py` ішінде орындалады;
- strict JSON schema қолданылады;
- бұл mock емес, нақты cloud AI.
- қажет болған жағдайда cloud backup қолданылады.

## 6. Нәтиженің дұрыстығы - 10 балл

Жоба:

- сұранысты өңдейді;
- құрылымды тест қайтарады;
- қате форматты анықтайды;
- quality report арқылы нәтижені тексереді;
- student mode арқылы практикалық тексеру жүргізе алады.

## 7. Қолданбалы құндылығы - 10 балл

Мұғалімге нақты пайдасы:

- уақыт үнемдейді;
- файлдан тест жасайды;
- student және teacher нұсқаларын бөледі;
- дайын тесті редакциялауға мүмкіндік береді;
- A/B/C варианттарын жасайды;
- question bank қолданады;
- аналитика береді.

## 8. Қате өңдеу және тұрақтылық - 5 балл

Жүйе келесі қателерді өңдейді:

- бос енгізу;
- API кілті жоқ;
- API қатесі;
- оқылмайтын файл;
- қолдау көрсетілмейтін формат;
- қате JSON;
- толық емес сұрақтар.
- student mode нәтижелерін сақтау;
- cloud sync істемесе локалды режимде жұмысты жалғастыру.

## 9. Жобаны таныстыруы - 10 балл

Қорғауға арналған материалдар дайын:

- `PROJECT_DEFENSE_KZ.md`
- `README.md`
- `CRITERIA_MAP_KZ.md`

Студент жобаның логикасын, интерфейсін, AI интеграциясын және педагогикалық пайдасын түсінікті көрсете алады.

## Қорытынды

Бұл жоба бағалау критерийлерін техникалық, интерфейстік және педагогикалық жағынан толық қамтиды. Дұрыс демонстрация жасалса, жоғары балл алуға толық негіз бар.
